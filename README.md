# Panda Planner 🐼
A clean and interactive monthly planner designed with a soft paper‑style UI, handwritten font, and a cute panda logo. The planner allows users to write daily notes, switch between months, and keep track of goals and thoughts — all inside a simple, friendly interface.

# Features ✨
Full monthly calendar layout
Navigate between months
Write notes inside each day
Two bottom sections for goals & general notes
Hand‑drawn aesthetic using Patrick Hand font
Responsive and easy to use

# Technologies Used 🛠️
HTML
CSS
JavaScript
